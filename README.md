#  RoboSpeaker 1.1

A simple Python-based text-to-speech app that speaks what you type.

##  Requirements

- Python 3.x  
- pyttsx3  
- espeak (for Linux)

Install with:

bash

sudo apt-get install espeak ffmpeg libespeak1
